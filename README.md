# Curiosify – Random Facts App

Curiosify is a small frontend application that fetches random facts from an external API and allows users to translate, copy, and save them to favorites.

## Live Demo

https://leogos.github.io/curiosify/

## Features

- Fetch random facts from an API
- Translate facts into Turkish
- Save favorite facts
- Copy fact text to clipboard
- Persist favorites with LocalStorage

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Fetch API
- LocalStorage API

## Project Purpose

This project was built to practice DOM manipulation, asynchronous JavaScript, API integration, and client-side state handling.
